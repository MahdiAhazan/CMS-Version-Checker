# Universal CMS Version Checker (WordPress, Joomla, Drupal)

یک اسکریپت شل (Bash Script) فوق‌العاده سریع، هوشمند و مستقل از نوع کنترل‌پنل که برای مدیران سرورهای لینوکس (cPanel, DirectAdmin, Plesk و سرورهای اختصاصی) توسعه یافته است. این ابزار به صورت سرور-محور (Server-Wide) تمام مسیرهای میزبانی را اسکن کرده و هسته‌های فعال سیستم‌های مدیریت محتوای محبوب را شناسایی می‌کند.

## 🚀 ویژگی‌های کلیدی و متمایز کننده
- **نمایش تجسمی پویا (Solid ASCII Banner):** مجهز به لوگوی متنی تراز شده و اختصاصی نام توسعه‌دهنده در محیط ترمینال.
- **پشتیبانی از تمامی نسل‌های جوملا:** اسکن دقیق متغیرهای قدیمی (J1.5 تا J3) و ساختارهای ثابت شیءگرای مدرن (J4، J5 و جوملا نسخه 6+).
- **تشخیص کامل سابدامین‌های مستقل:** به دلیل اسکن از روت اصلی لینوکس (`/`)، تمامی سایت‌های خارج از پوشه سنتی `public_html` نیز کشف می‌شوند.
- **تکنیک ضد ساب‌شل (No Subshell Loop):** رفع باگ Reset شدن متغیرها در لینوکس با متد `Process Substitution` جهت تضمین صحت آمار.
- **سرعت اسکن فوق‌العاده بالا (۱۰ برابر سریع‌تر):** استفاده از متد فیلتر هوشمند دایرکتوری (`-prune`) برای نادیده گرفتن فایل‌های حجیم سیستمی و ایمیل‌ها.
- **خروجی سه‌گانه همزمان (بدون حذف گزارش‌های قبلی):**
  1. **فایل گزارش متنی (.txt):** لاگ منظم و کلاسیک با برچسب زمان دقیق برای بررسی سریع.
  2. **فایل استاندارد برنامه‌نویسی (.json):** ذخیره ساختاریافته داده‌ها جهت اتصال مستقیم به APIها یا پلاگین‌های آینده سی‌پنل و دایرکت‌ادمین.
  3. **فایل وب مستقل (.html):** یک داشبورد گرافیکی، ریسپانسیو و مدرن با کارت‌های آماری رنگ‌بندی‌شده که در هر مرورگری بدون خطای امنیتی CORS باز می‌شود.

## 🛠️ راهنمای راه‌اندازی و اجرا در ۳ مرحله سریع

### ۱. ایجاد فایل اسکریپت روی سرور
با دسترسی کاربر ارشد (`root`) به محیط SSH سرور متصل شده و دستور زیر را وارد کنید:
```bash
nano cms_scanner.sh
```
*(کد ۴ مرحله‌ای اسکریپت را کپی کرده، داخل این فایل قرار دهید و با فشردن کلیدهای `Ctrl+O` و سپس `Ctrl+X` ذخیره کنید).*

### ۲. اعطای مجوز اجرایی به فایل
برای تبدیل فایل متنی به یک ابزار قابل اجرا در لینوکس، دستور زیر را بزنید:
```bash
chmod +x cms_scanner.sh
```

### ۳. اجرای سراسری اسکنر
حالا کافیست اسکریپت را اجرا کنید تا عملیات آنالیز لایه‌های سرور آغاز شود:
```bash
./cms_scanner.sh
```

## 📊 ساختار خروجی فایل‌ها و نحوه استفاده

پس از اتمام فرآیند اسکن، اسکریپت ۳ فایل مجزا با نام‌گذاری زمان‌دار (Timestamp) در پوشه روت جاری به شما تحویل می‌دهد:

- **`cms_report_YYYYMMDD_HHMMSS.txt`:** شامل لیست خط به خط پلتفرم‌ها و آدرس دقیق آن‌ها.
- **`cms_report_YYYYMMDD_HHMMSS.json`:** آرایه جیسون استاندارد، آماده برای توسعه پلاگین‌های گرافیکی وب در وب‌سرورها.
- **`cms_report_YYYYMMDD_HHMMSS.html`:** داشبورد گرافیکی اختصاصی پروژه. کافیست این فایل را روی کامپیوتر خود دانلود کرده و با دو بار کلیک در هر مرورگری (Chrome, Firefox, Edge, Safari) باز کنید تا آمار خلاصه و جدول تفکیک‌شده را به صورت بصری مشاهده کنید.
